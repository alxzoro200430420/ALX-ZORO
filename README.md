# Login System with PHP/MySQL

## Installation

1. Import database from `install.php`
2. Copy `config/database.example.php` to `config/database.php`
3. Update database credentials in `config/database.php`
4. Default admin: admin@example.com / admin123

## Features

- Secure login/registration
- Password hashing
- Session management
- MySQL database

## Requirements

- PHP 7.4+
- MySQL 5.7+
